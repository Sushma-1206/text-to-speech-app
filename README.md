# 🎧 Web Speech API - Text To Speech App

A clean, modern, responsive front-end web application that converts text into spoken audio using the native browser **Web Speech API**.

## 🚀 Features
* **Custom Text Input:** Type or paste any text you want to convert to speech.
* **Character Counter:** Real-time character counting as you type.
* **Voice Selection:** Dynamic dropdown that automatically loads all available system voices on your device.
* **Audio Controls:** Adjustable sliders to change the **Speed (Rate)** and **Pitch** of the voice.
* **Visual Status Indicator:** A pulsing status dot that changes states (`Ready`, `Speaking...`, `Stopped`, `Error`).

## 🛠️ Built With
* HTML5
* CSS3 (Custom Variables, Flexbox, CSS Grid)
* JavaScript (ES6+, Web Speech API)

## 📦 How to Run Locally
1. Clone this repository or download the files.
2. Open `index.html` directly in your browser (or use a local server extension like **Live Server** in VS Code).
3. Type your text, pick a voice, and hit **Speak**!
